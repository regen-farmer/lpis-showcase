RegenMaps

Explore European agricultural parcel data (LPIS) with different mapping libraries. Choose your preferred technology below.

MapLibre GL JS

High-performance WebGL-based maps with smooth vector tile rendering, globe projection, and rich interactivity.

OpenLayers

Full-featured, battle-tested mapping library with extensive format support and powerful vector tile capabilities.

Leaflet

Lightweight and mobile-friendly mapping with the popular leaflet-maplibre-gl plugin for vector tile support.

About the Data

This showcase demonstrates access to the Regen Farmer LPIS (Land Parcel Identification System) API, which provides agricultural field boundaries from multiple European countries as vector tiles.
